# LibreOffice-Reverse-Polish-Notation-RPN-macro
LibreOffice Calc macro implementing a simple Reverse Polish Notation (RPN) calculator for the Polish locale. Numbers use the decimal comma, tokens are separated by semicolons or spaces. Supports +, −, ×, /, ^, and basic text ops: concatenation (+) and repeat (text * n). Includes a worksheet function and dialog-driven macros. Shortcut keys enabled.!
